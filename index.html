교육 콘텐츠 개요

1. Apache Pinot 소개

Pinot의 기본 개념, 주요 특징, 활용 사례를 알아봅니다.

2. 아키텍처

Pinot의 주요 컴포넌트와 데이터 흐름 아키텍처를 이해합니다.

3. 데이터 모델링

테이블 유형, 스키마 정의, 데이터 타입을 배웁니다.

4. 데이터 처리 흐름

오프라인, 실시간, 하이브리드 데이터 처리 방법을 탐구합니다.

5. 데이터 사용 흐름

쿼리 처리, 필터링, 집계, JSON 데이터 처리 방법을 배웁니다.

6. 성능 최적화 전략

인덱싱, 쿼리 최적화, 사전 집계로 성능을 향상시킵니다.

7. 운영 관리

클러스터 운영, 백업, 스케일링, 모니터링 방법을 학습합니다.

8. 실습 예제

실제 데이터 저장, 인덱스 설정, 쿼리 예제를 실습합니다.
